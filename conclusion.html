Conclusion du projet KDou

Ce projet nous a permis de mieux comprendre et exploiter les données de KDou pour en tirer des informations stratégiques.

Dans ce projet, où nous avons pu examiner et organiser toutes les données de l'entreprise KDou dès le début, nous avons appris à quel point il est important de savoir comment les données d'une entreprise sont stockées et dans quel ordre elles sont conservées.

Nous avons également compris l'importance des données d'une entreprise dans le chemin qu'elle suivra pour atteindre les objectifs qu'elle s'est fixés pour son avenir. Les données enregistrées montraient à quel point un objectif était réaliste ou ce qui devait changer pour atteindre cet objectif.

Dans ce projet, où nous avons également eu l'occasion de comparer les logiques Access et Excel, nous avons pu observer les différences entre les deux applications et connaître les avantages offerts par l'application Access.

Dans ce projet, dans lequel nous avons appris comment manipuler les données, nous avons également appris quels types d'erreurs nous pouvons rencontrer lors de la saisie des données et quelles précautions nous devons prendre contre ces erreurs.

Enfin, en utilisant le langage logiciel SQL, nous avons appris à manipuler les données dont nous disposions de la meilleure façon possible, et grâce à celles-ci, nous avons pu réaliser les analyses et les résultats dont nous avions le plus besoin.

Nous pensons que ce projet nous a non seulement appris quelques langages de programmation, mais nous avons également beaucoup appris sur le fonctionnement d'une entreprise, ses objectifs et sa situation économique, ainsi que sur l'importance des données.

Nous avons observé les points suivants :

  • Amélioration de la qualité des données.
  • Développement d’indicateurs pertinents pour la prise de décision.

Les outils utilisés, notamment SQL et le low-code, ont permis de simplifier les processus d’analyse.
